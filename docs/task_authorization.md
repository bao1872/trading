## 任务授权流程

如需添加新任务，必须遵循以下流程：

### 1. 明确任务信息
在 `scheduler.py` 头部注释中明确列出：
- 任务名称
- 任务函数
- 触发器类型和参数
- 执行时间
- 功能描述
- 配置项名称

### 2. 添加配置项
在 `.env` 文件中添加对应的配置项：
```bash
ENABLE_NEW_TASK=true  # 新任务开关
```

### 3. 实现任务函数
在 `scheduler.py` 中实现任务函数：
```python
def new_task():
    """新任务 - 已授权"""
    logger.info("开始执行新任务")
    try:
        # 任务逻辑
        logger.info("任务执行完成")
    except Exception as e:
        logger.error(f"任务执行失败：{e}", exc_info=True)
        raise
```

### 4. 注册任务
在 `main()` 函数中显式注册：
```python
ENABLE_NEW_TASK = os.getenv('ENABLE_NEW_TASK', 'false').lower() == 'true'

if ENABLE_NEW_TASK:
    scheduler.register_task(
        name='新任务',
        func=new_task,
        trigger='cron',
        hour=10,
        minute=0,
        day_of_week='mon-fri',
    )
```

### 5. 更新文档
更新 `scheduler.py` 头部的任务列表注释和本文档。
